### Installation 👀

1. Download the newest [release](https://github.com/swiftly-solution/swiftly_discordutilities/releases).
2. Perform a drag-and-drop installation.

### Features 🧐

- Chat Logs (All & Team)
- Connect/Disconnect Logs
- Map Change Logs
- Game Events Logs
- Report/Calladmin Command
